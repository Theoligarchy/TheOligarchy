About

What is The Oligarchy?

The Oligarchy is an independent research platform focused on criminology, psychology, and politics. It explores crime, human behaviour, and systems of power through articles and analysis.

The goal is to simplify complex topics through research, observation, and critical thinking. Not sensationalism. Just structured analysis made accessible.

Philosophy is not a separate category here. It runs through everything, from how we think about crime, to the psyche, to political systems.

    Follow these steps exactly to get theoligarchy.in live. Takes about 30–45 minutes total.

    Step 1
    Download this site's code
    Click the artifact's download/copy button in Claude. Save the file as index.html on your desktop.
    Step 2
    Create a free GitHub account
    Go to github.com → Sign up. Use any email. Verify your account.
    Step 3
    Create a new repository
    Click the + icon → New repository. Name it theoligarchy. Set it to Public. Click Create.
    Step 4
    Upload your file
    Inside the repository, click Add file → Upload files. Drag your index.html there. Commit changes.
    Step 5
    Enable GitHub Pages
    Go to Settings → Pages. Under Source, select main branch, / (root). Save. Your site goes live at theoligarchy.github.io/theoligarchy
    Step 6
    Connect your GoDaddy domain
    In GitHub Pages settings, add custom domain: theoligarchy.in. Then go to GoDaddy → DNS Settings → Add these 4 A records pointing to GitHub's IPs: 185.199.108.153 185.199.109.153 185.199.110.153 185.199.111.153. Also add a CNAME record: www → yourusername.github.io. DNS takes up to 24 hours.
    Step 7 — Posting Articles
    How to post after going live
    Right now posts save in your browser only. To make posts visible to everyone: Write your article here in Admin → open the site on your own computer → go to Admin → write and publish → then download the updated index.html (it has the posts baked in via localStorage export) and re-upload to GitHub. OR — ask Claude to add a Google Sheets integration so posts sync automatically. That's the next upgrade.
    Important Note
    The admin password is visible in the page source. For a public site this is fine for now — just don't share your password publicly. The real fix is a backend (Supabase/Firebase) which is a future upgrade.
